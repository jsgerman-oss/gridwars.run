# Security Policy

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories:

→ <https://github.com/jsgerman-oss/gridwars.run/security/advisories/new>

We aim to acknowledge reports within 7 days and to coordinate disclosure on a timeline appropriate to the severity.

## Supported versions

GridWars.run is pre-1.0 and ships a single supported version (the latest tagged release on `main`). Security fixes will land on `main` and be tagged as patch releases.

| Version | Supported |
|---|:---:|
| `v0.1.x` | ✅ |
| `< v0.1` | ✗ |

## Out of scope

- Vendored Evennia bugs — please file upstream at <https://github.com/evennia/evennia/security/advisories>
